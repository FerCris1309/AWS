Resumo – Visão Geral da AWS

Este documento apresenta uma visão geral do curso "AWS Cloud Foundations", ministrado por Alexsandro Lechner, Arquiteto de Soluções AWS.

OBJETIVO GERAL
O curso tem como objetivo desenvolver habilidades técnicas relacionadas à AWS, desde conceitos fundamentais até tópicos avançados, como computação sem servidor, bancos de dados gerenciados, segurança e automação. É voltado para profissionais com conhecimento básico em tecnologia, familiaridade com programação e interesse em Cloud Computing.

CONTEÚDO PROGRAMÁTICO
Módulo 1: Introdução à AWS e Conceitos Básicos
- História da AWS e origem do nome “Amazon”.
- Estrutura e expansão global da AWS.
- Infraestrutura: Regiões e Zonas de Disponibilidade.
- Modelo de Negócio baseado em pagamento por uso (OPEX).
- Diferença entre os modelos IaaS, PaaS e SaaS.

PRINCIPAIS CONCEITOS
- A AWS é o maior provedor de serviços em nuvem do mundo, oferecendo mais de 200 serviços.
- Sua infraestrutura cobre 105 zonas de disponibilidade em 33 regiões globais, com expansão contínua.
- O modelo de pagamento flexível e a variedade de serviços (computação, armazenamento, banco de dados, machine learning, IoT) são diferenciais competitivos.
- O curso enfatiza práticas “hands-on”, permitindo aos alunos aplicarem na prática os conceitos aprendidos.

CONCLUSÃO
A AWS se destaca pela confiabilidade, segurança e inovação em computação em nuvem, sendo essencial para profissionais de tecnologia que desejam se atualizar no mercado.
